# Static Resources Directory

此目录用于存放静态资源文件。

## 说明

前端应用已迁移到独立的 Vue 3 项目中，位于项目根目录的 `vue-code/` 文件夹。

如需访问前端应用，请：

1. 进入 `vue-code/` 目录
2. 运行 `npm install` 安装依赖
3. 运行 `npm run dev` 启动开发服务器
4. 或运行 `npm run build` 构建生产版本

## 前端技术栈

- Vue 3
- TypeScript
- Element Plus
- Vite
